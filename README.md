# RUBA-issuer-docs
Официальная документация эмитента ЗАО "РУБА КЕЙ ДЖИ" стейблкоина RUBA 

Этот репозиторий содержит официальную документацию эмитента стейблкоина RUBA, включая: 
- Whitepaper, подготовленный для целей первичного выпуска стейблкоина на территории Республики Кыргызстан
- Программа внутреннего контроля в целях ПФТД/ЛПД
- Политика использования файлов cookie
- Политика конфиденциальности и обработки персональных данных

Документация будет обновляться и дополняться по мере развития проекта и появления новых требований.
